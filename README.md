# Enterprise Sim

Generate a fake-but-realistic enterprise organization and all the office-work artifacts it
would produce over a configurable time window — departments, teams, people (with job
descriptions, locations, calendars), projects, and the documents/presentations/schedules
those people create, in native formats (`.docx`, `.pptx`, `.md`, `.json`).

**Start here:** [`PLAN.md`](./PLAN.md) — the full design plan and open questions.

Status: pre-implementation. Plan drafted; build not yet started.
